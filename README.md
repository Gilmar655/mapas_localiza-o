# Acompanhamento de Obras MV Leste / Norte

Site atualizado com a programação do período **25/05/2026 a 30/05/2026**.

## Base de dados

- Arquivo analisado: `Programação_25-30-maio.xlsx`
- Aba utilizada: `Programação_Semana`
- Registros carregados: 82
- Obras com coordenadas: 74
- Sem coordenadas: 8

## Publicação no GitHub Pages

1. Acesse o repositório `mapas_localiza-o`.
2. Clique em **Add file > Upload files**.
3. Envie o arquivo `index.html` para a raiz do repositório.
4. Clique em **Commit changes**.
5. Aguarde a atualização do GitHub Pages.

O arquivo `index.html` está na raiz deste pacote ZIP.
